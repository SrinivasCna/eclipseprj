Collection Framework
~~~~~~~~~~~~~~~~~~~~~~

https://docs.oracle.com/javase/8/docs/api/

Advantages of Collection Framework
1. Reduces programming effort
2. Increases performance

interface Iterable
interface Collection extends Iterable

interface List extends Collection
A. class ArrayList implements List
B. class LinkedList implements List

interface Queue extends Collection
A. class PriorityQueue implements Queue

interface Set extends Collection
interface NavigableSet extends Set
A. class HashSet implements Set
B. class LinkedHashSet implements Set
C. class TreeSet implements NavigableSet

class AbstractMap implements interface Map
class HashTable extends AbstractMap
class HashMap extends AbstractMap
